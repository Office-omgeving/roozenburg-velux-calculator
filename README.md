# Roozenburg VELUX calculator

Statische prototype-calculator voor Roozenburg VELUX-aanvragen.

## Lokaal draaien

Open `index.html` rechtstreeks in de browser, of start een eenvoudige lokale server:

```bash
python3 -m http.server 4173
```

Daarna is de calculator beschikbaar op `http://localhost:4173/`.
